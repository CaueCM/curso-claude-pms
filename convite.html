B E M   V I N D O   À   A R E N A C A S H

Você foi contratado como PM da ArenaCash.

A ArenaCash é uma fintech criada pela Product Arena. Ela não existe de verdade — mas os problemas que você vai resolver aqui são os mesmos que surgem toda semana no trabalho real de um PM.

Antes do curso, você recebeu o ArenaCash Kit: memo estratégico do CEO, transcrições das conversas com CFO, CMO e CTO, dados de produto e financeiros do quarter anterior, tickets de atendimento, design system e template oficial de apresentação.

São os mesmos materiais que um PM receberia no primeiro dia numa empresa real. A diferença é que aqui você não vai só ler e analisar — vai usar o Claude para trabalhar com eles como PM de verdade: entender o negócio, priorizar iniciativas, criar documentos, apresentar para a liderança, prototipar, codar e publicar.

A jornada completa do dia: você começa configurando o Claude e montando seu ambiente de PM. Analisa os dados do quarter, prioriza uma iniciativa e valida com os C-levels. Essa iniciativa vira apresentação para a diretoria, depois protótipo visual, depois mapeamento de eventos — e termina com o app publicado online com uma URL real.

"Todo o curso acontece dentro do Claude. Sem slides. Sem sair da plataforma."
A N T E S   D E   C O M E Ç A R

Pré-requisitos

Faça isso antes do dia do curso. Cada item abaixo tem um motivo — sem ele, alguma parte do dia não funciona. Separe cerca de 30 minutos pra concluir tudo.

1
Conta Claude paga Pro, Max ou Team · sem isso o curso não começa
Obrigatório

O Claude gratuito tem limite de mensagens e não acessa o Cowork nem os plugins. O curso inteiro roda no Claude pago.

Qual plano escolher

  • Pro — funciona pro curso. É o mínimo necessário.
  • Max — recomendado se você quiser aproveitar sem se preocupar com limites. Durante um dia intenso como o curso, o Pro pode atingir o limite de mensagens. O Max tem limites muito mais altos. Veja a diferença entre os planos →
  • Team — se sua empresa já tem conta Team, peça ao administrador pra te adicionar.

Se ainda não tem conta: claude.ai/upgrade pra assinar.

2
Claude Desktop App interface principal do curso · só existe no desktop
Obrigatório

O Cowork — que usamos toda a manhã — só existe no Desktop. No navegador e no celular ele não aparece.

  1. Baixe o instalador em claude.ai/download (Mac ou Windows)
  2. Abra o arquivo baixado e siga a instalação normalmente
  3. Faça login com a mesma conta Claude paga do passo anterior
  4. Verifique que aparece a opção Cowork no menu lateral — se aparecer, tá tudo certo
3
Claude Code usado na parte da tarde do curso
Obrigatório

Claude Code é uma ferramenta que você instala no terminal e que permite criar protótipos, sites e scripts em conversa com o Claude. É a ferramenta da parte da tarde.

Pra instalar, você vai precisar de duas coisas na ordem abaixo:

3a · Node.js (versão 18 ou mais recente)

Node.js é o motor que roda o Claude Code. Você provavelmente já tem — verifique antes de instalar.

  1. Abra o Terminal (Mac: Cmd + Espaço → "Terminal") ou PowerShell (Windows: tecla Windows → "PowerShell")
  2. Digite node --version e pressione Enter
  3. Se aparecer algo como v20.x.x ou v22.x.x, você já tem. Pule pro 3b.
  4. Se não aparecer nada: nodejs.org/en/download e baixe a versão LTS pro seu SO. Instalador padrão.

3b · Claude Code

Com o Node instalado, no mesmo terminal cole o comando abaixo e pressione Enter:

npm install -g @anthropic-ai/claude-code

Aguarde (pode demorar 1-2 minutos). Depois verifique:

claude --version

Se aparecer o número da versão, tá instalado. Documentação: docs.anthropic.com/pt/docs/claude-code.

4
Git o Claude Code depende dele por baixo dos panos
Obrigatório

Git é um sistema de controle de versão. O Claude Code usa por baixo dos panos pra gerenciar arquivos e mudanças — sem ele, o Code pode dar erro na inicialização.

  • Mac: Git já vem instalado na maioria dos casos. Verifique com git --version no Terminal. Se não tiver, o próprio Mac vai oferecer instalar automaticamente — aceite.
  • Windows: Verifique com git --version no PowerShell. Se não tiver, baixe em git-scm.com/download/win (opções padrão, não precisa mexer em nada).
5
Kit ArenaCash baixado e descompactado · é o material do curso
Obrigatório

O kit contém todos os documentos da empresa fictícia que vamos usar no dia: memo estratégico, organograma, dados de Q1, transcrições dos C-levels, design system e guia de eventos.

↓   Baixar ArenaCash-Kit.zip ou ver arquivos individuais abaixo ↓

Depois de baixar:

  • Descompacte em algum lugar fácil de achar (ex: Desktop ou Documentos)
  • Renomeie a pasta pra ArenaCash-PM
  • Verifique que consegue ver arquivos .md, .csv, .xlsx e .pdf dentro dela
6
Conta Google / Gmail usado no curso pra extrair seu tom de voz
Recomendado

Vamos conectar o Gmail ao Cowork pra extrair o seu tom de voz de escrita a partir dos seus e-mails. Funciona com qualquer conta Google.

  • Se você tem Gmail, não precisa fazer nada — só lembrar o login
  • Se preferir não usar a conta pessoal, crie uma conta separada só pro curso em accounts.google.com
  • Se não quiser usar Gmail de jeito nenhum, tudo bem — o instrutor mostra um caminho alternativo pra criar o tom de voz manualmente
