# Claude para PMs — Do Chat ao Código

Site navegável da ementa do curso **Claude para PMs** da Product Arena.

- Single-page HTML estático, sem build step
- Identidade visual ArenaCash (laranja #F26522, Poppins/Inter)
- 8 blocos · 15 exercícios · agenda clicável · sidebar sticky

## Rodando local

Basta abrir `index.html` no navegador. Nenhuma dependência.

## Atalhos

- Tecla `E` abre/fecha todos os exercícios de uma vez
- Clique em qualquer item da agenda para pular ao bloco
